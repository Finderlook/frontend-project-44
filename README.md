## Brain-games is a simple set of terminal mini-games.

### How to install games:

1. Open console and execute:
2. Clone the repo

  ```sh
  git clone git@github.com:Finderlook/frontend-project-44.git
  ```
3. To install project:

   ```sh
   make install
   ```
      ```sh
   make publish
   ```
      ```sh
   npm link
   ```
